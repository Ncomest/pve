# Статы боссов (данные JSON)

В `bosses.json` и `resources.json` у объекта `stats` задаются:

| Поле         | Смысл |
|-------------|--------|
| `power`     | Сила атаки |
| `armor`     | Броня (очки; снижение урона по формуле брони) |
| `chanceCrit`| Шанс крита (доля 0..1) |
| `evasion`   | Уклонение (доля 0..1) |
| `accuracy`  | Меткость (доля 0..1) |
| `critDefense` | Защита от крита (доля 0..1) |

Поле **скорости** у боссов не используется; интервал автоатаки босса в бою фиксирован в коде.
