# Таксономия паттернов способностей

Документ описывает каталог паттернов способностей для единого языка при дизайне и реализации. Новая способность задаётся как комбинация **основного действия** и **дополнительных эффектов** (ресурс, бафф, дебафф, сейв и т.д.).

---

## 1. По основному действию

| Группа | Паттерны | Примеры в коде |
|--------|----------|-----------------|
| **Чистый урон** | Мгновенный удар (один тик) | Атака, Коварный удар |
| **Урон + DoT** | Мгновенный удар + кровотечение/яд по тикам | Кровотечение, Ядовитый укус |
| **Чистое лечение** | Восстановление HP | Лечение |
| **Чистый бафф** | Только усиление себя (сила, крит, уклонение и т.д.) | Усиление атаки, Ускользание |
| **Чистый дебафф** | Только ослабление цели (броня, замедление и т.д.) | — |
| **Защита / сейв** | Уклонение от следующего удара, блок, снижение входящего урона | Уворот, Блок, Плащ теней |
| **Контроль** | Прерывание, стан и т.д. | Укор (interrupt) |
| **Мобильность** | Скорость, отбег и т.д. | Молниеносная вспышка |
| **Ресурс** | Генерация/расход комбо (или другого ресурса) | Генераторы/финишеры «Клинок и Яд» |

---

## 2. Комбинированные паттерны

- **Удар + генерация ресурса** — урон и комбо-поинты (или другой ресурс).
- **Удар + усиление другой способности** — урон + стаки/бафф под следующий скилл (Потрошение, Коварство).
- **Удар + бафф на себя** — урон + крит/сила/уклонение на время.
- **Удар + дебафф на цель** — урон + срез брони/замедление/уязвимость.
- **Удар + DoT** — урон + кровотечение/яд.
- **Удар + сейв** — урон + краткий щит/уклонение (редко).
- **Лечение + бафф** — хил + небольшой бафф (реген, броня и т.д.).
- **Чистое ослабление цели** — только дебафф без урона.
- **Чистое усиление себя** — только бафф (сила, крит, уклонение, скорость).
- **Сейв** — уворот от следующей атаки / блок особых / снижение урона на время.

Классы (например, «Клинок и Яд») — набор способностей, каждая соответствует одному или нескольким таким паттернам.

---

## 3. Список паттернов для справки (~20–30)

1. Мгновенный урон (без доп. эффектов).
2. Мгновенный урон + генерация ресурса (комбо и т.д.).
3. Мгновенный урон + усиление другой способности (стаки, бафф под скилл).
4. Мгновенный урон + бафф на себя.
5. Мгновенный урон + дебафф на цель.
6. Мгновенный урон + DoT.
7. Мгновенный урон + DoT с проком (бафф при тике).
8. Только лечение.
9. Лечение + бафф на себя.
10. Только бафф на себя (сила, крит, уклонение, скорость).
11. Только дебафф на цель (броня, замедление).
12. Урон + сейв (редко).
13. Сейв: уклонение от следующей атаки.
14. Сейв: блок особых атак.
15. Сейв: снижение входящего урона на время.
16. Сейв: повышение уклонения на время.
17. Контроль: прерывание способности цели.
18. Мобильность: бафф скорости.
19. Расход ресурса (финишер): чистый урон.
20. Расход ресурса: урон + DoT.
21. Расход ресурса: урон с масштабированием по стакам/ресурсу.

Дополнительно можно вводить: урон + два дебаффа, бафф на себя + сейв, условный эффект при крите и т.д.

---

## 4. Крит атакующих способностей

У каждой **атакующей** способности можно задать:

- **chanceCrit** (0..1) — дополнительный шанс крита; складывается с общим шансом крита персонажа (и с временными баффами, например от «Размашистого удара»).
- **critMultiplier** (число) — сила крита: множитель урона при критическом ударе. По умолчанию **1.5** (150%): при обычном ударе 10 ед. крит даёт 15 ед.

Если поля не указаны: шанс крита = только от персонажа, множитель крита = 1.5.

---

## 5. Использование при добавлении способностей

- При добавлении новой способности указать **паттерн** (номер из списка или комбинацию).
- В коде при использовании системы эффектов способность задаётся списком эффектов (`effects: AbilityEffect[]`); новый паттерн = новая комбинация эффектов или один новый тип эффекта + один обработчик.
- Для атакующих способностей при необходимости задать **chanceCrit** и **critMultiplier** (см. п. 4).
